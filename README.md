# نظام حجز مواعيد – Frontend (Portfolio Project)

Full-Stack demo application – حجز مواعيد احترافي

**Live Demo:** https://booking-frontend-portfolio.vercel.app/booking  
**Backend API Docs:** https://booking-backend-portfolio-production.up.railway.app/docs
**Backend Repo:** https://github.com/khadrx/booking-backend-portfolio

### المميزات الرئيسية
- تقويم عربي كامل (أيام وشهور مترجمة)
- أوقات متاحة ديناميكية (يمنع الحجز المكرر)
- نموذج حجز مع validation + toast notifications (Sonner)
- لوحة تحكم لعرض جميع الحجوزات
- دعم Dark/Light mode
- RTL 100% + Responsive

### Tech Stack
**Frontend**
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui
- Sonner (notifications)
- date-fns + Arabic locale

**Backend** (منفصل)
- FastAPI
- SQLAlchemy + PostgreSQL
- Pydantic v2
