# Booking Frontend – نظام حجز مواعيد

Full-Stack demo application (frontend only)  
Next.js 15 + TypeScript + Tailwind + shadcn/ui + Sonner

### المميزات
- تقويم عربي (date-fns/locale/ar)
- أوقات متاحة ديناميكية
- نموذج حجز مع validation + toast notifications
- دعم dark/light mode
- Responsive & RTL

### Tech Stack
- Next.js 15 (App Router)
- shadcn/ui
- Sonner (notifications)
- date-fns + locale/ar
- Tailwind CSS

### Run locally
npm install
npm run dev

Open: http://localhost:3000/booking
